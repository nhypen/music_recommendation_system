# 🎵 Music Recommendation System

A simple command-line **Music Recommendation System** written in Python.  
This project allows users to browse songs, get recommendations based on preferences, and manage their own music database.

## Author: 
nhypen

## 🚀 Features

- 📄 Show all songs in the database  
- 🎯 Get music recommendations based on:
  - Genre
  - Mood
  - Language  
- 🔍 Search songs by artist  
- ➕ Add your own songs  
- ❌ Remove songs  
- 💾 Save songs to a file (`songs.json`)  
- 📂 Load songs from file automatically  
- 🎲 Get a random recommendation  

---

## 🛠️ Technologies

- Python 3
- JSON (for data storage)
- Standard libraries:
  - `json`
  - `os`
  - `random`
